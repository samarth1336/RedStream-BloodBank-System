# RedStream – Seamless Blood Supply Chain Management System

A smart blood bank management platform using AI, IoT, and Blockchain for real-time donor and inventory management.

## Features
- Real-time blood inventory tracking
- IoT-based monitoring
- Blockchain-secured records
- AI-based donor matching
- Emergency blood request management

## Technologies Used
- Python
- Blockchain
- IoT
- SQL
- Machine Learning

## Conference Publication
Published and presented at MLIP-2025 International Conference.

## Applications
- Hospitals
- Blood banks
- Emergency healthcare systems
